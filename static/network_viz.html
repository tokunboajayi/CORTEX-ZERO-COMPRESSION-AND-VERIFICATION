How SIREN Works

SIREN (Sinusoidal Representation Network) uses periodic sine activations instead of ReLU to represent continuous signals like images.

Given (x, y) coordinates, it outputs the RGB color at that position. The entire image is encoded in the network weights.

Compression: A 256x256 image (192KB) can be compressed to ~200KB of network weights.

class SirenNet(nn.Module): # Input: (x, y) coordinates # Output: RGB values SineLayer(2 -> 256, omega=30, is_first=True) SineLayer(256 -> 256, omega=30) SineLayer(256 -> 256, omega=30) SineLayer(256 -> 256, omega=30) Linear(256 -> 3) # RGB output
